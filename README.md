# Zonefile validation

## Installation

* Pipenv https://pypi.org/project/pipenv/

```sh
brew install pipenv
```

### Execute

```sh
$ pipenv shell
$ pipenv install
$ make build
$ make console/csv

Usage: python check.py [FILE] [MY_DOMAIN]
    [FILE] zone file
    [MY_DOMAIN] treat this domain as 1st party
```

Output may look like this
